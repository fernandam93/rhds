# rhds

## To recreate the mamba environment used in the practical:

```
git clone https://github.com/fernandam93/rhds.git
cd rhds
mamba env create -f environment.yml
mamba activate rhds
Rscript install.r
```

## To set up config on your system:
- Create a config.env file based on the config-template.env template, with the right directories
